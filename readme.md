# 三国强手棋（临时名称）

小时候玩的一个桌游《武将疯云录》的实现。

#### 如何运行

	npm install
	gulp generate-web
	gulp generate-sass
	npm start

### 首页
[http://localhost:18080](http://localhost:18080)

### api 文档
